#  Projeto Terraform AWS EC2

Este repositório contém a automação para provisionar uma infraestrutura básica na AWS utilizando **Terraform**. O objetivo foi criar uma instância de servidor e configurar as regras de firewall (Security Group) automaticamente.

##  O que foi criado:
* **1x Instância EC2:** Servidor Ubuntu configurado via código.
* **1x Security Group:** Firewall liberando as portas 22 (SSH) e 80 (HTTP).

## Tecnologias Utilizadas:
* **Terraform** (Infraestrutura como Código)
* **AWS** (Cloud Provider)
* **Git/GitHub** (Versionamento e Portfólio)

##  Desafios Resolvidos:
Durante o projeto, superei erros de permissão de IAM, ajuste de tipos de instância (t2 vs t3) e configuração de tokens de segurança para o GitHub.
